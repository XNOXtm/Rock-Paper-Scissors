This is a simple Rock Paper Scissor game.

Made while learing JavaScript. With minimal UI styling. Learned a lot about DOM methods and js fundamentals.

HOW TO PLAY:
    1. Select you preferred choice out of the Three. <br>
    2. Play until one of you(computer or user) scored 5 points. <br>
    3. Referesh the page to Restart the game. <br>
